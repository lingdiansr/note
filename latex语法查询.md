| gamma      | `\gamma`      | *γ*      |

| epsilon    | `\epsilon`    | *ϵ*      |

| zeta       | `\zeta`       | *ζ*      |
| eta        | `\eta`        | *η*      |
| theta      | `\theta`      | *θ*      |
| vartheta   | `\vartheta`   | *ϑ*      |
| iota       | `\iota`       | *ι*      |
| kappa      | `\kappa`      | *κ*      |
| lambda     | `\lambda`     | *λ*      |
| mu         | `\mu`         | *μ*      |
| nu         | `\nu`         | *ν*      |
| xi         | `\xi`         | *ξ*      |
| pi         | `\pi`         | *π*      |
| varpi      | `\varpi`      | *ϖ*      |
| rho        | `\rho`        | *ρ*      |
| varrho     | `\varrho`     | *ϱ*      |
| sigma      | `\sigma`      | *σ*      |
| varsigma   | `\varsigma`   | *ς*      |
| tau        | `\tau`        | *τ*      |
| upsilon    | `\upsilon`    | *υ*      |
| phi        | `\phi`        | *ϕ*      |
| varphi     | `\varphi`     | *φ*      |
| chi        | `\chi`        | *χ*      |
| psi        | `\psi`        | *ψ*      |
| omega      | `\omega`      | *ω*      |
